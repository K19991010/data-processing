split-txt.py是一个标注数据集划分文件
可以将xml、yolo格式的数据集按比例划分为训练集、验证集、测试集
  原始文件夹：
    mydata：
      --JPEGImages
      --Annotations
  生成文件夹：
    datasets：
      --images:
        --train
        --val
        --test
      --labels:
        --train
        --val
        --test
      --train.txt
      --val.txt
